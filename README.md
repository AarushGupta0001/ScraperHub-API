# ScraperHub API

Multi-source data aggregation backend built with ASP.NET Core.

## Features
- Product price scraping
- YouTube channel discovery & video tracking
- Esports tournament scraping
- EF Core database integration

## Tech Stack
- ASP.NET Core (.NET 8)
- EF Core
- HtmlAgilityPack
- YouTube Data API
